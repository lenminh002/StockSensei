## Added
- Structured JSON response contract for AI output.
- Deterministic terminal UI blocks for cards, tables, bars, range bars, sparklines, and news.
- Visual builder tools for stock snapshots, comparisons, charts, and news.
- Animated single-line status feedback while the agent runs.
- Interactive slash-command completion with `/help`, `/models`, and `/quit`.
- `/quit` command.

## Changed
- Reworked agent output from freeform markdown to structured renderable blocks.
- Improved terminal visuals with brighter colors and more consistent card/bar layouts.
- Shifted comparison output toward cards, bars, and cleaner tables.
- Simplified streaming feedback so internal tool details are hidden from users.

## Removed
- Thesis feature and all `/thesis ...` commands.
- Local thesis storage module and related UI/render logic.
- Thesis references from help text, command dropdowns, packaging, and README.
