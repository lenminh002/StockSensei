## Added
- `/clear` command — clears the terminal screen and resets conversation memory, starting a fresh session without restarting the app.
- `/clear` listed in slash-command autocomplete dropdown and `/help` output.
- `providers.py` — extracted `PROVIDER_PRESETS`, `LANGCHAIN_PROVIDER_MAP`, and `get_langchain_provider` out of `config.py` into their own module.

## Changed
- Added `/quit` command while keeping `exit`, `quit`, and `q` as quick exit aliases.
- Refactored `ui_blocks.py`: extracted `_fmt_market_cap`, `_fmt_price`, `_fmt_change`, `_fmt_pe` helpers to eliminate duplicated formatting logic and shorten long lines.
- `make_price_comparison_block` rewritten as a list comprehension using the new format helpers.
- `build_news_visual` in `tools.py`: extracted `symbol` variable to break a 120+ char return line.
- `make_summary_comparison_block` now renders actual valuation and 52-week comparison fields instead of company profile metadata.
